# 📱 Responsive Landing Page

A modern, fully responsive landing page built with HTML5 and CSS3. Designed to adapt perfectly across desktop, tablet, and mobile devices.

## ✨ Features

- **Fully Responsive**: Works on all screen sizes using Flexbox, Grid & Media Queries
- **Modern UI**: Clean layout with Hero section, Features, CTA, and Footer
- **Mobile-First Approach**: Optimized for mobile devices first
- **Fast & Lightweight**: No frameworks or external libraries used
- **Cross-Browser Compatible**: Works on Chrome, Firefox, Safari, Edge

## 🛠️ Tech Stack

- **HTML5** - Semantic Structure
- **CSS3** - Styling, Flexbox, Grid, Animations
- **Responsive Design** - Media Queries
